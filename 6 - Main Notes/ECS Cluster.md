# ECS Cluster
An ECS Cluster is a logical grouping of tasks or services

EC2 Launch Type
	If running tasks of services using the EC2 launch type, a cluster is also a grouping of container instances

Capacity Providers
	A cluster is a logical grouping of capacity providers

A cluster can be a combination of Fargate & EC2 Launch Types
