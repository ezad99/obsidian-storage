# Obsidian Quick Start Guide

## Creating a new note:
1. `Ctrl + o`: Create a new note, and put in a new title,
	1. If note already exists it will show in the dropdown
2. `Shift + Enter`: Creates the note
3. `Ctrl + T`: To add a template to the note
